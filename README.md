📜 The Path of Eloquence – Nahjul Balagha Compiler

Our mini-project focuses on employing modern compiler tools to extract, analyze, and structure the profound wisdom contained in select statements from Nahjul Balagha, a cornerstone of Islamic literature. By leveraging lexical analysis, parsing, and semantic analysis, we aim to organize the teachings into a clear, structured format suitable for further study or digital use.

📖 About Nahjul Balagha

Nahjul Balagha (“The Path of Eloquence”) is a revered collection of sermons, letters, and sayings attributed to Ali ibn Abi Talib, compiled by Sharif Razi in the 10th century. It is widely considered one of the most significant works in Islamic literature.

⚙️ Compilation Process

The project follows a compiler-inspired workflow to process the text:

Tokenization – Using Lex, we break the text into meaningful tokens (words, phrases, punctuation).

Parsing – Using Yacc, we analyze the grammatical structure of statements and identify relationships between elements.

Semantic Analysis – Ensures correctness of meaning, resolves ambiguities, and organizes statements in a structured format.
